# Security Policy

VibeGesture is intended to run locally on macOS.

## Supported versions

- Current stable baseline: `stable-20260418`
- Supported runtime platform: macOS 14+

## Reporting a security issue

If you discover a security or privacy issue, please report it privately through GitHub Security Advisories or by opening a private maintainer report.

Please include:
- A clear summary of the issue
- The affected version or commit
- Steps to reproduce
- Any relevant logs or screenshots

## Privacy notes

- The app uses the default camera for local hand pose processing
- The current implementation does not include cloud-based inference
- Camera and Accessibility permissions are required for the runtime to function
